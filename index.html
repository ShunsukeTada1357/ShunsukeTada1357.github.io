Shunsuke Tada

I am Shunsuke Tada (多田駿介 in Japanese) studying mathematics for (mainly) Topological Data Analysis. I am currently interested in persistence modules over finite posets from a view of homological algebra, and computations of their structures. GitHub

Computation:
[1] Code for visualizing Gabriel quiver of END(G), where G is all the direct sum of the non-iso interval modules. This code is the implementation of Proposition 6.8. of the paper "EXACT STRUCTURES FOR PERSISTECE MODULES" by B. Blanchette, T. BrüStle, and E.J.Hanson. See arXiv:2308.01790 for more details.

[2] A code for calculating compressed multiplicity defined in the paper "Approximation by interval-decomposables and interval resolutions of persistence modules" by H. Asashiba, E. G. Escolar, K. Nakashima, M. Yoshiwaki. See arXiv:2207.03663 Definition 5.1. The code is available, and implemented in a software pmgap.

[3] A computation of bipath persistent homology. See Bipathposets. It is an implementation of Algorithm 3 in our paper Bipath persistence.
